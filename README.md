## ‚ú® Key Features

### üîÑ Continuous Job Monitoring
- Multi-platform scraping (LinkedIn, Indeed, Glassdoor, ZipRecruiter, Dice)
- Smart deduplication (URL, content hash, company+role+date)
- Configurable regions, roles, and filters

### üß† Intelligent Matching (0-100 Score)
- **Required skills match**: Does your resume have must-have skills?
- **Preferred skills match**: Bonus points for nice-to-haves
- **Project alignment**: Relevant GitHub/Kaggle projects boost your score
- **ATS simulation**: Keyword density and formatting checks
- **Experience matching**: Years of experience vs job requirements

### üõ°Ô∏è Scam Detection
Flags suspicious postings based on:
- Unrealistic salary for seniority level
- Payment requests in job description
- Free email domains for company contact
- Missing or invalid company information
- Recruiter red flags

### üìö Project Intelligence
Automatically searches and adds relevant projects from:
- **GitHub** (public repos matching tech stack)
- **HuggingFace** (ML models and datasets)
- **Kaggle** (competitions and notebooks)
- **ArXiv** (research papers)

Labels autogenerated projects as such in metadata (ethical transparency).

### ‚ú® Resume Optimization
- **JD-tailored rewriting**: Optimizes keywords while maintaining truthfulness
- **ATS-friendly formatting**: Passes applicant tracking systems
- **Version control**: Every tailored resume is saved
- **Project insertion**: Adds relevant projects to showcase skills

### üíå Smart Cover Letters
Generate personalized letters with 6 personality styles:
- **Professional**: Formal, business-appropriate
- **Friendly**: Warm, approachable
- **Technical**: Detailed, engineering-focused
- **Direct**: Concise, to-the-point
- **Creative**: Unique, storytelling
- **Relocation-friendly**: Highlights flexibility

### ü§ñ Auto-Application
- **Playwright-powered** browser automation
- **Multi-step form handling** (application wizards)
- **File upload support** (resume, cover letter, portfolio)
- **CAPTCHA prompting**: Notifies you to solve (never bypasses)
- **Screenshot audit trail**: Captures every step
- **Sandbox mode**: Test before going live

### üìä Analytics Dashboard
Real-time insights:
- Total scraped, matched, applied
- Scams detected, duplicates avoided
- Match score distribution
- Interview/offer rates
- Top skills demand
- Company statistics
- Timeline charts

### üîî Notifications
- **Email**: Daily digests and instant alerts
- **Webhooks**: Slack, Discord, Zapier
- **In-app**: Real-time notifications

## üöÄ Quick Start

### Prerequisites
```bash
- Python 3.11+
- Node.js 18+
- PostgreSQL 15+ (or Neon/Supabase account)
- OpenAI/Anthropic API key
```

### 1. Clone Repository
```bash
git clone https://github.com/yourusername/career-agent.git
cd career-agent
```

### 2. Backend Setup
```bash
cd backend

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Install Playwright browsers
playwright install chromium

# Set environment variables
cp .env.example .env
# Edit .env with your database URL, API keys, etc.

# Run database migrations
alembic upgrade head

# Start backend
uvicorn app.main:app --reload --port 8000
```

Backend will be running at: `http://localhost:8000`

### 3. Frontend Setup
```bash
cd ../frontend

# Install dependencies
npm install

# Set environment variables
cp .env.example .env.local
# Edit .env.local with VITE_API_URL=http://localhost:8000

# Start frontend
npm run dev
```

Frontend will be running at: `http://localhost:5173`

### 4. Create User & Start Searching
```bash
# Register a user
curl -X POST http://localhost:8000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "you@example.com",
    "password": "secure-password",
    "full_name": "Your Name"
  }'

# Visit http://localhost:5173
# Login and configure your job search preferences
# Upload your base resume
# Start monitoring!
```

## ‚öñÔ∏è Legal & Ethics

### ‚úÖ What Career Agent Does
- Respects robots.txt on all scraped sites
- Never bypasses CAPTCHA or site protections
- Never fabricates employment history or skills
- Labels autogenerated projects as such
- Provides user control and manual approval options
- Encrypts PII at rest
- Offers data deletion and opt-out

### ‚ùå What Career Agent Does NOT Do
- Bypass authentication or paywalls
- Scrape sites that explicitly forbid it
- Submit applications without user approval (in default mode)
- Fabricate work experience or education
- Spam job boards or employers

### User Responsibilities
- Provide accurate information
- Review and approve applications before submission
- Comply with job board terms of service
- Use ethical judgment
